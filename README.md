# Lock Forge
A lightweight CLI tool to securely encrypt and decrypt local files using strong AES-256 encryption. Built with Bash and OpenSSL, designed to be simple, fast, and usable directly from the terminal.

## Overview
Lock Forge protects sensitive files by encrypting them with industry-standard cryptography. It uses OpenSSLâ€™s AES-256 with PBKDF2 key derivation to ensure strong password-based security.

## Features
- AES-256 encryption using OpenSSL  
- Password-based key derivation (PBKDF2)  
- File encryption and decryption  
- Hidden password input  
- Wrong password detection  
- Lightweight and dependency-minimal
